# My Lambda Project

Dự án Lambda của tôi.

## Cấu trúc thư mục

| Thư mục/Tệp               | Mô tả                                    |
|---------------------------|------------------------------------------|
| `src/`                     | Mã nguồn chính của Lambda               |
| `src/__init__.py`          | Khởi tạo thư mục (nếu cần thiết)         |
| `src/lambda_function.py`   | Tệp Python chứa hàm Lambda chính        |
| `src/utils.py`             | Các hàm phụ trợ khác (nếu có)            |
| `requirements.txt`         | Các thư viện cần thiết cho dự án        |
| `.gitignore`               | Các tệp và thư mục không muốn đưa vào git |
| `README.md`                | Tài liệu hướng dẫn, mô tả dự án         |
| `test/`                    | Thư mục chứa các bài kiểm tra           |
| `test/__init__.py`         | Khởi tạo thư mục (nếu cần thiết)         |
| `test/test_lambda_function.py` | Các bài kiểm tra cho Lambda Function |
| `.env`                     | Các cấu hình cho Lambda, các biến môi trường |
